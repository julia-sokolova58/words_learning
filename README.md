# Извлечение лексики из книги

Веб-приложение на Django для поиска отрывка в книге, извлечения немецких слов (глаголов и существительных) с помощью spaCy и YAKE, а 
также получения переводов из базы данных. Для каждого слова доступны страницы с грамматикой (парсинг verbformen.ru) и синонимами.

## Что мы сделали

- Поиск отрывка в книге по первым и последним 5 словам
- Лемматизация и извлечение глаголов/существительных (spaCy, YAKE)
- Вывод слов с переводами из базы `GermanWord` (скраулили на основе кода из гитхаб. Приложен)
- Отдельные страницы для грамматики и синонимов
- Грамматика парсится с verbformen.ru (но мы не смогли проверить работоспособность функции. на остальной код не влияет)
- Синонимы через предобученную модель word2vec (German Word2Vec)
